# 상품 설명 페이지 에디터 기획서

---

## 핵심 컨셉

> 사용자는 빈 캔버스 앞에서 막막함을 느낀다.
> 잘 설계된 **블록 단위 템플릿**을 제공하고,
> 그 위에서 **자기 입맛에 맞게 다듬는** 경험을 제공한다.

### 구조 원리

```
블록 (Block)
 └─ 하나의 섹션 단위. 레이아웃 + 기본 요소(이미지/텍스트/도형)가 세트로 구성됨
     예) 메인 배너 블록, 특징 3단 블록, 스펙 표 블록 ...
     블록은 세로 방향으로 쌓이며 순서 변경 가능

템플릿 (Template)
 └─ 여러 블록을 조합하여 하나의 완성된 페이지를 구성한 것
     예) 식품 템플릿 = [메인 배너] + [원재료 블록] + [영양성분 블록] + [주의사항 블록]
```

### 캔버스 & 출력 영역 모델

```
← 작업 공간 전체 (무한) ───────────────────────────────────────┐
│                                                               │
│      ┌─────────────────────────────┐  ← 출력 영역            │
│      │  B01 메인 배너              │     (고정 폭 1200px,     │
│      ├─────────────────────────────┤      블록 높이 합산)     │
│      │  B02 좌이미지-우텍스트      │                          │
│      ├─────────────────────────────┤                          │
│      │  B05 특징 3단               │                          │
│      └─────────────────────────────┘                          │
│                                                               │
│  [이미지]  ← 출력 영역 밖으로도 자유 이동 가능               │
│                                                               │
└───────────────────────────────────────────────────────────────┘

PNG/JPG 저장 시 → 출력 영역(점선 안쪽)만 캡처
출력 영역 밖의 요소는 저장에 포함되지 않음
```

### 사용자 여정

```
막막함
  │
  ▼
[1] 템플릿 갤러리에서 가장 비슷한 것 선택
  │   └─ 이미 완성된 블록 조합이므로 그럴듯해 보임
  │
  ▼
[2] 에디터에서 내 상품에 맞게 다듬기
  │   ├─ 이미지 교체
  │   ├─ 텍스트 수정
  │   ├─ 블록 순서 변경 / 추가 / 삭제
  │   └─ 개별 요소 자유 편집 (위치/크기/색상/폰트)
  │
  ▼
[3] 완성 → PNG/JPG 저장
  │
완성
```

## 프로젝트 개요

- **목적**: 쿠팡 상품 등록용 설명 이미지를 웹에서 제작
- **결과물**: PNG 또는 JPG 이미지
- **기준 사이즈**: 가로 1200px (쿠팡 권장 기준)

---

## 블록 종류 (Block Library)

각 블록은 **레이아웃 + 기본 요소**가 세트로 구성되어 있어 선택 즉시 완성형으로 보임

| 블록 ID | 블록명 | 레이아웃 설명 |
|---------|--------|---------------|
| B01 | 메인 배너 | 전체 배경 이미지 + 중앙 제목 텍스트 |
| B02 | 좌이미지-우텍스트 | 이미지 50% / 특징 설명 텍스트 50% |
| B03 | 우이미지-좌텍스트 | B02의 좌우 반전 |
| B04 | 특징 2단 | 아이콘+텍스트 2열 나열 |
| B05 | 특징 3단 | 아이콘+텍스트 3열 나열 |
| B06 | 전체 이미지 | 이미지 100% (텍스트 오버레이 가능) |
| B07 | 스펙 표 | 항목/값 2열 표 구성 |
| B08 | 갤러리 2열 | 이미지 2개 나란히 |
| B09 | 갤러리 3열 | 이미지 3개 나란히 |
| B10 | 텍스트 중심 | 제목 + 본문 텍스트 블록 |
| B11 | 주의사항 | 아이콘 + 텍스트 리스트 |
| B12 | 브랜드 배너 | 브랜드명 + 슬로건 + 배경색 |

## 템플릿 목록 (Template Gallery)

각 템플릿은 위 블록들의 조합

| 템플릿 ID | 이름 | 카테고리 | 블록 구성 |
|-----------|------|----------|-----------|
| T01 | 기본형 | 공통 | B01 → B02 → B05 → B07 → B11 |
| T02 | 감성형 | 뷰티/패션 | B01 → B06 → B03 → B10 → B12 |
| T03 | 정보형 | 식품/건강 | B01 → B04 → B07 → B08 → B11 |
| T04 | 이미지 중심형 | 공통 | B01 → B06 → B09 → B02 → B12 |
| T05 | 미니멀형 | 뷰티 | B01 → B10 → B06 → B03 → B11 |

## 편집 범위 정의

### 블록 레벨 편집
- 블록 순서 변경 (위/아래) — 세로 방향으로 쌓이는 구조
- 블록 추가 (블록 라이브러리에서 삽입)
- 블록 삭제
- 블록 높이 조절

### 요소 레벨 편집 (캔버스 위 개별 요소)
- 이미지: 업로드 후 캔버스 어디든 자유 이동 / 크기 조절 / 회전 / 불투명도
- 텍스트: 내용 수정 / 폰트 / 크기 / 색상 / 정렬
- 도형: 색상 / 크기 / 불투명도
- 배경: 블록별 배경색 / 배경 이미지 교체
- **요소는 출력 영역 밖으로도 자유롭게 이동 가능**
- **저장 시에는 출력 영역 안쪽만 캡처됨**

## 기능 명세

### 캔버스 인터랙션
- 클릭: 요소 선택
- 더블클릭: 텍스트 편집 진입
- 드래그: 요소 이동
- 핸들 드래그: 크기 조절 / 회전
- Shift+클릭: 다중 선택
- Ctrl+C/V: 복사/붙여넣기
- Delete: 삭제
- Ctrl+Z/Y: 실행취소/다시실행

### 자동저장
- 편집 중 로컬스토리지에 자동저장
- 새로고침 후에도 작업 유지

### 내보내기
- PNG 저장 (고해상도)
- JPG 저장 (품질 선택 가능)
- 파일명 직접 지정

## UX 원칙

1. **막막함 제거**: 빈 캔버스 없음. 항상 완성형 템플릿에서 시작
2. **점진적 편집**: 텍스트/이미지 교체만으로도 충분히 완성 가능한 수준의 템플릿
3. **스냅 가이드**: 요소 정렬 시 가이드라인으로 정렬 보조
4. **즉각적 피드백**: 편집 내용이 캔버스에 실시간 반영
5. **결과물이 예뻐야 한다**: 사용자가 별다른 감각 없이도 완성도 있는 결과물이 나와야 함

## 디자인 품질 기준 (핵심)

> 기능보다 중요한 것은 **결과물의 완성도**
> 사용자가 폰트·배치를 잘 몰라도 템플릿 자체가 이미 완성도 있어야 함

### 폰트 시스템

- 한국어 웹폰트 위주로 선별 제공 (직접 고른 폰트만 노출)
- 추천 폰트 목록 (카테고리별 분류):

| 용도 | 폰트 후보 |
|------|-----------|
| 제목 (임팩트) | Noto Sans KR Black, 나눔스퀘어 ExtraBold, SUIT ExtraBold |
| 본문 (가독성) | Noto Sans KR Regular, Pretendard Regular |
| 감성/뷰티 | Nanum Myeongjo, Gowun Batang |
| 귀엽고 캐주얼 | 둘기마요, Jua |

- 폰트 선택 UI는 **미리보기 포함** (텍스트가 실제 폰트로 표시됨)
- 추천 폰트 조합 제공 (제목용 + 본문용 세트)

### 색상 톤 시스템

> 색상을 개별로 고르면 조합이 망가짐 → **톤 세트 단위**로 묶어서 제공
> 사용자는 톤만 선택하면 배경색·강조색·텍스트색이 한번에 바뀜

- 각 톤은 **배경색 + 강조색 + 텍스트색 + 서브텍스트색** 4가지가 세트로 구성
- 사용자는 톤을 고르기만 하면 됨 (개별 색상 선택 불필요)
- 컬러피커는 고급 옵션으로만 제공 (기본 노출 안 함)

| 톤 이름 | 배경 | 강조 | 텍스트 | 분위기 |
|---------|------|------|--------|--------|
| 클린 화이트 | #FFFFFF | #222222 | #111111 | 깔끔, 범용 |
| 소프트 그레이 | #F5F5F5 | #333333 | #222222 | 모던, 심플 |
| 딥 다크 | #1A1A1A | #FFFFFF | #F0F0F0 | 프리미엄, 고급 |
| 웜 베이지 | #FAF6F0 | #8B6F47 | #3D2B1F | 자연, 친환경 |
| 소프트 핑크 | #FDF0F3 | #D4607A | #3D1A24 | 뷰티, 여성 |
| 민트 프레시 | #F0FAF8 | #2A9D8F | #1A3D38 | 식품, 건강 |
| 선샤인 옐로 | #FFFBF0 | #E9A800 | #3D2E00 | 활기, 식품 |
| 네이비 트러스트 | #F0F4FA | #1B3A7A | #0D1F45 | 신뢰, 전문 |

- 템플릿 선택 시 어울리는 기본 톤이 자동 적용됨
- 원클릭으로 다른 톤으로 전체 교체 가능
- 톤 미리보기: 선택 전 실제 색상 조합이 스와치로 표시됨

### 테마 시스템 (톤 + 폰트 세트)

> 톤과 폰트를 따로 고르면 조합이 어색해질 수 있음
> **테마** = 톤 + 폰트 조합을 미리 검증된 세트로 묶어 제공

| 테마명 | 톤 | 제목 폰트 | 본문 폰트 | 대상 상품 |
|--------|----|-----------|-----------|-----------|
| 클린 모던 | 클린 화이트 | SUIT ExtraBold | Pretendard Regular | 생활용품, 전자 |
| 프리미엄 다크 | 딥 다크 | 나눔스퀘어 ExtraBold | Noto Sans KR | 고급 소비재 |
| 내추럴 오가닉 | 웜 베이지 | Gowun Batang | Noto Serif KR | 친환경, 식품 |
| 뷰티 감성 | 소프트 핑크 | Nanum Myeongjo | Pretendard Light | 뷰티, 화장품 |
| 헬시 그린 | 민트 프레시 | SUIT Bold | Pretendard Regular | 건강식품 |
| 활기 비비드 | 선샤인 옐로 | 나눔스퀘어 Black | Noto Sans KR | 식품, 스낵 |

- 사용자는 테마만 고르면 됨 → 색상 + 폰트 한번에 적용
- 개별 조정은 고급 옵션으로 제공

### 레이아웃 품질

- 블록 내 여백(패딩)이 기본으로 충분히 확보됨
- 텍스트와 이미지 간 간격이 시각적으로 안정적인 비율로 설계
- 블록 기본 높이는 쿠팡 상세페이지 관행 기준으로 설정

### 템플릿 디자인 기준

- 각 템플릿은 실제 쿠팡 상위 노출 상품 레이아웃을 참고하여 설계
- 텍스트 기본값은 그냥 읽어도 자연스러운 예시 문구로 채움
- 이미지 슬롯에는 플레이스홀더 이미지 제공 (교체 전에도 그럴듯해 보임)
